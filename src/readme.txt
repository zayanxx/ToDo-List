Income Expense Calculator Task

A simple web application to track income and expenses.
It allows users to add, edit, and delete entries.
The total income, expenses, and net balance.

Features
Add income expense entries.
Edit or delete existing entries.
Filter entries by All...Income....Expense.
Persist data using local storage.
Responsive design for desktop and mobile.

Technologies Used
HTML
Tailwind CSS (for styling)
JavaScript

JavaScript is used:
use add method get value
get input value from input field
to store data in localStorage
use filter method to filter data
and edit data delete data from localStorage
